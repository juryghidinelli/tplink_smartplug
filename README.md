# TplinkSmartplug

Ruby porting of Python project https://github.com/softScheck/tplink-smartplug

## Installation

Add this line to your application's Gemfile:

    gem 'tplink_smartplug'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tplink_smartplug
